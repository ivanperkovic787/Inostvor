# ByVendor — native CAD exporti

Ovdje idu DXF datoteke izvezene IZRAVNO iz komercijalnih CAD alata, po poddirektoriju
alata (npr. `AutoCAD/`, `SolidWorks/`, `Fusion360/`, `QCAD/`). Uz svaku datoteku
navesti alat + verziju + postavke exporta u ovom README-u. Svaka datoteka dobiva
vlastiti regresijski test.
